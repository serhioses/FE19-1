Как выполнять ДЗ

Для выполнения домашних работ создать директорию javascript в папке views. Затем в ней создавать директории соответственно ДЗ (howework-1, homework-2, homework-N). Затем в нужной homework-N создать файл index.html в котором подключить файл скрипта перед </body>. Файл скрипта хранить в папке js внутри текущего урока. В файле скрипта обязательно использовать 'use strict'. Перед выполнением ДЗ проверить файл урока script.js и homework-sample.js, в которых пример оформления домашек.
